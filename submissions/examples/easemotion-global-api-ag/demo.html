The programmatic Modal API manages focus trapping, scroll prevention, and restores focus back to the triggering element on dismiss.

window.EaseMotion.Modal.close('id')
Developer Tip: Open your browser Console and type EaseMotion.Modal.open('diagnostics-modal') to test controls programmatically.

Programmatic API Details

  • SPA Routing Compatible: Eliminates hash-based router conflicts (e.g. #my-modal causing page jumps in React/Next.js/Vue routers) by managing display classes directly.
  • Accessible Focus-Trapping: Keyboard focus loops dynamically inside the modal when active, restoring focus back to the triggering element on close (compliant with WAI-ARIA).
  • Centralized Scroll Reveal Observer: Centralizes multiple elements to a single IntersectionObserver, preventing memory footprint leaks.
